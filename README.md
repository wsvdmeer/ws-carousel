ws-carousel
===========

Minimal jquery carousel

How to use:

Html:
```html
<div class="carousel">
  <ul class="items">
    <li>item 1</li>
    <li>item 2</li>
    <li>item 3</li>
  </ul>
</div>
```

Jquery:
```html
$(".carousel").wscarousel();
```

Defaults:
<ul>
<li>start: 0</li>
<li>auto:true</li>
<li>speed:500</li>
<li>delay:1000</li>
<li>time:8000</li> 
<li>autoHide:true</li>
<li>showNavigation:true</li>
</ul>

Notes:
Pagination and next/previous button are added by the script.
